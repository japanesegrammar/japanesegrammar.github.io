    Conditional - と (to)

    Conditional - と「to」form

    The と「to」form can be used to express the result/event which is predictable and unavoidable due to an action.

    Info

    You cannot use と 「to」to express the speaker's wishes, judgement, permission, hopes, requests

    Conjugation Rule for と「to」

    Sentence: [Condition] と [Result]

    use present tense short form + と


    Example A
  1. ここを押（お）すと、ドアが 開（ひら）きます。
    2. Press here to open the door.

    押（お）す
    —
    attach と
    →
    押（お）すと

    Example B
  2. メアリーさんが 国（くに）に帰（かえ）ると 寂（さび）しくなります。
    3. When Mary returns home, we will be lonely.
    帰（かえ）る
    —
    attach と
    →
    帰（かえ）ると

    Info

    In this example, we are expressing the cause-effect relationship within the event


    Example C

  3. 私（わたし）は子供（こども）の時（とき、）冬（ふゆ）になると風邪（かぜ）をひきました。
    4. When I was a child, I caught a cold whenever winter arrived.

    なる
    —
    attach と
    →
    なると

    Info

    Even though we have to use present tense to attach with と「to」, we still can express the past event.


    Example D: I-Adjective at the second clause

  4. 秋（あき）になると木（き）が赤（あか）くなります。
    5. The trees turn red in the fall.

    なる
    —
    attach と
    →
    なると

    Example E: Na-Adjective at the second clause
  5. 夜（よる）になると町（まち）が静（しず）かになります。
    6. The town becomes quite at night.

    なる
    —
    attach と
    →
    なると

    Info

    For both example D and E, take note that we use an adjective (い「i」 and な「na」Adjective respectively) at the second clause. This kinds of sentence express a change in the content; something has turned/became due to a first clause.

    Vocabulary

    1. 私（わたし） ⇌ I; me​
    2. 押（お）す ⇌ to push; to press​
    3. 開（ひら）く ⇌ to open
    4. ドア ⇌ door
    5. 国（くに） ⇌ country; hometown
    6. 帰（かえ）る ⇌ to return; to come home
    7. 寂（さび）しい ⇌ lonely
    8. 子供（こども） ⇌ child​
    9. 時（とき） ⇌ time; hour; moment
    10. 冬（ふゆ） ⇌ winter
    11. 風邪（かぜ） ⇌ common cold
    12. ひく ⇌ to catch a cold
    13. 秋（あき） ⇌ autumn; fall​
    14. 木（き） ⇌ tree
    15. 赤（あか）い ⇌ red; crimson
    16. 夜（よる） ⇌ evening; night​
    17. 町（まち） ⇌ town​
    18. 静（しず）か ⇌ quiet
